# JDBC-Practice

This repository contains JDBC (Java Database Connectivity) programs demonstrating database operations using Java and MySQL.

## Topics Covered

- Database Connection
- Insert Records
- Retrieve Records
- Update Records
- Delete Records
- PreparedStatement
- ResultSet Handling
- Exception Handling

## Technologies Used

- Java
- JDBC
- MySQL

## Purpose

This project is created for learning and practicing JDBC concepts, database connectivity, and SQL operations in Java applications.
